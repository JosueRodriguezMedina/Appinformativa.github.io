Aplicaciones

Estas aplicaciones estan programadas para poder crear, administrar, modificar, y trabajar bases de datos, estos son solo algunos ejemplos, puedes encontrar mas en nuestro apartado de aplicaciones.

SqlServer

Es un sistema de gestión de base de datos relacional, desarrollado por la empresa Microsoft. Utilizado para manipular y recuperar datos, crear tablas y definir relaciones entre ellas.

AmazonAurora

Amazon Aurora es un servicio de base de datos relacional desarrollado y ofrecido por Amazon Web Services que fue lanzado en octubre de 2014.

PostgreSQL

Es un sistema de gestión de bases de datos relacional orientado a objetos y de código abierto, similar a la BSD o la MIT.
