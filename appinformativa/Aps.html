Aplicaciones

Estas aplicaciones estan programadas para poder crear, administrar, modificar, y trabajar bases de datos, estos son solo algunos ejemplos, puedes encontrar mas en nuestro apartado de aplicaciones.


Abajo encontraras una amplia variedad de aplicaciones junto a sus instaladores

SqlServer

Es un sistema de gestión de base de datos relacional, desarrollado por la empresa Microsoft. Utilizado para manipular y recuperar datos, crear tablas y definir relaciones entre ellas. Instalador

AmazonAurora

Amazon Aurora es un servicio de base de datos relacional desarrollado y ofrecido por Amazon Web Services que fue lanzado en octubre de 2014. Instalador

PostgreSQL

Es un sistema de gestión de bases de datos relacional orientado a objetos y de código abierto, similar a la BSD o la MIT. Instalador

MariaDB

MariaDB es un sistema de gestión de bases de datos derivado de MySQL con licencia GPL.
Instalador

Oracle Autonomous Database

Oracle Autonomous Database proporciona una base de datos fácil de usar y totalmente autónoma
Instalador

Azure Cosmos DB

Azure Cosmos DB es un servicio de base de datos multimodelo distribuido globalmente ofrecido por Microsoft Azure. Instalador

Google Cloud Datalab

Cloud Datalab es una herramienta de Google Cloud para la exploración avanzada de datos.
Instalador

Tableau Server

Implemente con una flexibilidad inigualable, proporcione datos confiables, Integre información basada en los datos donde trabaja y mas con Tableau Server. Instalador
